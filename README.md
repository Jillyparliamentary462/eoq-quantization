## 🧭 What this is

EOQ-quantization is a Windows-ready way to use smaller LLM files with strong quality. It uses entropy-optimal quantization, which helps reduce model size while keeping perplexity close to FP16. That means you can store and run models in less space without giving up much quality.

This project is for end users who want to download a release, open the app or package, and use it on a Windows PC.

## 🪟 What you need

- A Windows 10 or Windows 11 PC
- Enough free disk space for the model files
- A stable internet connection for the download
- A modern CPU or GPU, based on the release you choose
- Admin access if Windows asks for permission during setup

For the best experience, use a machine with at least 16 GB of RAM. Smaller models may run on less, but more memory gives smoother results.
